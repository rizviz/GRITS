# Roadmap

## v0.1 goals

- establish GRITS as an open lifecycle governance and runtime assurance framework
- ship immediate adoption artifacts for OpenClaw-style deployments
- define the minimum managed-object and lifecycle model for agents and LLM apps
- publish a small but working scan/score/remediate workflow

## Immediately after v0.1

- NemoClaw compatibility and hardening guidance
- ownership and accountability refinement
- promotion and recertification playbook
- more example scorecards and event streams
- drift signal catalog

## Later candidates

- MCP/tooling profile
- sector overlays
- deeper delegation and transitive trust model
- machine-readable control catalog and schemas
- CI-friendly validation pack
